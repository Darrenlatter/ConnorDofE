Welcome to my site!

Week 1 – Getting Started (HTML basics) “I learned how to structure a webpage using headings and paragraphs. At first, it looked confusing, but once I saw the tags working in the browser, it made sense. I liked seeing text I wrote appear on the page.” Week 2 – Adding Style (CSS basics) “This week I used CSS to change colours and fonts. It was fun experimenting with different looks. I found it tricky to get the background and text to contrast well, but I solved it by testing on different screens.” Week 3 – Images and Links “I learned how to insert images and make links clickable. It was great to link to my favourite website. The hardest part was resizing images so they didn’t look stretched.” Week 4 – Navigation Menus “I built a simple navigation bar. I struggled with the menu alignment, but after looking at examples, I got it to line up properly. It feels more like a ‘real’ website now.” Week 5 – Layouts with CSS “This week I worked with page layouts. I learned how to create columns and adjust spacing. It took a few tries to stop things overlapping, but I managed to fix it by using margins.” Week 6 – Responsive Design “I made the site work on mobile. Testing on my phone was really useful. At first, the text was too big, but I changed the CSS so it looked better. Now the site works on both desktop and phone.” Week 7 – Adding a Contact Form “I created a simple form with boxes for name and email. It was interesting to see how forms can be used. The hardest part was making it look neat, but CSS helped me fix it.” Week 8 – Embedding Media “I embedded a YouTube video and some audio. I had never done this before, but it was exciting to see video play inside my website. I felt proud when it worked.” Week 9 – Interactivity with JavaScript “I learned how to add a button that changes the background colour. It was tricky because I kept making mistakes with the code, but after checking carefully, I got it working.” Week 10 – Improving Design “I polished the website by adding borders, shadows, and better spacing. I really like how it looks more professional now. It felt good to see how much my design has improved.” Week 11 – Publishing the Website “I published my site online using GitHub Pages. At first, I couldn’t get the files to upload correctly, but once I followed the steps, it worked. I felt proud to share my link with friends.” Week 12 – Final Review “I looked back at my whole project. I learned a lot about HTML, CSS, and JavaScript. The most enjoyable part was adding videos and making the site interactive. I now feel more confident using IT skills.”
